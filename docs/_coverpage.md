# mdfx

> Markdown effects: Unicode text styling and UI components

**CLI Tool:**
```bash
cargo install mdfx-cli
```

**Library:**
```toml
[dependencies]
mdfx = "1.0"
```

- **19 Unicode Styles** - mathbold, script, fraktur, fullwidth, and more
- **Template Syntax** - {{mathbold}}TEXT{{/mathbold}} transforms automatically
- **Multi-Backend Rendering** - shields.io URLs or local SVG files
- **UI Components** - Dividers, swatches, status indicators, tech badges
- **Frame System** - Visual frames with gradient, solid, and dashed styles
- **Badge System** - Circled numbers, negative squares, parentheses
- **174 Tests** - Comprehensive test coverage with confidence
- **Cross-Platform** - macOS • Linux • Windows • WSL2
